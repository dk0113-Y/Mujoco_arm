# Mujoco_arm

MuJoCo-based Franka Panda arm control and grasping experiments.

## Current progress

- MuJoCo 3.10.0 environment configured
- Franka Panda model loaded from MuJoCo Menagerie
- Joint position control verified
- Gripper open/close control verified
